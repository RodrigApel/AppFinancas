# AppFinancas
Aplicativo de finanças pessoais para endividados
Aplicativo com interações em linguagem coloquial para que Pessoas Fisicas possam ter maior controle financeiro visando pagamento de dividas em atraso.
Usuario informara renda e gastos essenciais e dividas em atraso.
O aplicativo ira apresentar o limite de gastos para reestabelecer saude financeira.
App indicará quais dividas devem ser pagas primeiro.
